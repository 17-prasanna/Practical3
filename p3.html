Demonstrating Inline, Internal, and External CSS

This is a paragraph with inline CSS.
Inline CSS is applied directly to an HTML element using the style attribute. It allows developers to define styles for a specific element without using external or internal stylesheets. This method is useful for quick styling or when only minor changes are required. However, it is not recommended for large-scale projects as it can make the HTML code cluttered and difficult to maintain.

This paragraph is styled using internal CSS.
Internal CSS is defined within a style tag inside the head section of an HTML document. It allows for styling multiple elements within the same page without the need for an external stylesheet. This method is beneficial when working on a single HTML page that does not require reusable styles. However, if multiple pages need the same styles, it is better to use external CSS to maintain consistency and reduce redundancy.

This paragraph is styled using external CSS.
External CSS is written in a separate .css file and linked to an HTML document using the link tag. This method is the most efficient and scalable way to apply styles, as it allows for reusability and keeps the HTML code clean. External CSS is widely used in web development for maintaining a consistent design across multiple pages. By modifying a single CSS file, developers can update the styling of an entire website.
